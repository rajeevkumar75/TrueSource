About TrueSource

Built for trust in the age of generative AI

TrueSource is a multi-model authenticity platform that helps journalists, educators, researchers, and everyday users verify whether images, videos, and text are genuine or synthetically generated.

The project combines computer vision and natural language processing into a single web experience — train your own models, deploy them locally, and analyze content without sending data to third-party APIs.

  • ✓ Open-source friendly — runs entirely on your machine
  • ✓ Image & video via ResNet18 frame classifier
  • ✓ Text via fine-tuned DistilBERT
  • ✓ Free to use with no account required
Vision

Make deepfake detection accessible to everyone, not just ML researchers.

Tech Stack

PyTorch · Flask · Transformers · OpenCV · Custom trained weights

Use Cases

News verification, academic integrity, social media moderation, research.
