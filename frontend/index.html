Capabilities

Three models. One platform. Complete coverage.

Purpose-built neural networks analyze every dimension of synthetic content with state-of-the-art accuracy.

ResNet18 CNN

Image Deepfake Detection

Binary classification using a fine-tuned ResNet18 convolutional neural network. Supports JPEG, PNG, WEBP, BMP with adjustable confidence thresholds.

  • Upload file or paste image URL
  • Adjustable fake probability threshold
  • Full probability breakdown chart
  • Export results as JSON
Scan an image →
Frame-Level Analysis

Video Deepfake Detection

Intelligent frame extraction and multi-frame aggregation using mean/max pooling. Analyzes temporal consistency across up to 64 frames per video clip.

  • Uniform frame sampling
  • Mean, Max, or Mean-Max aggregation
  • Frame count & timing stats
  • Support for MP4, AVI, MOV, WEBM
Scan a video →
DistilBERT NLP

AI Text Detection

Fine-tuned DistilBERT transformer trained to distinguish human-written from AI-generated content. Supports single analysis and batch paragraph mode.

  • Single & batch paragraph analysis
  • Upload .txt files directly
  • Live word & character counter
  • Tunable AI probability threshold

Architecture

How TrueSource works

01

Upload or Link

Drop an image/video file, paste a public URL, or type/upload text. All data stays local — nothing is ever sent externally.

02

Local Inference

Your Flask backend fetches model weights from your private HuggingFace repo and runs PyTorch inference on your hardware (CPU or CUDA).

03

Instant Results

Confidence scores, probability bars, a clear verdict, and a full exportable JSON report — all rendered in milliseconds.
